# uploadbot
clone of anydlbot of telegram
---

#### Installation

```sh
virtualenv -p python3 VENV
. ./VENV/bin/activate
pip install -r requirements.txt
cp config.sample.py config.py
--- EDIT config.py values appropriately ---
python bot.py
```
